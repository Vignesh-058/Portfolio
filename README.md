# Professional Portfolio

This is a modern, responsive, and highly aesthetic professional portfolio built for **Vignesh R**, an Aspiring Full Stack Developer. It features a dark-mode, glassmorphism design with an interactive, animated background.

## Features

- **Modern UI**: Clean, glassmorphism components with neon pink, indigo, and violet accents.
- **Dynamic Backgrounds**: CSS-powered floating blobs and moving starry particle animations.
- **Micro-Animations**: Scroll-reveal animations using `IntersectionObserver`.
- **Responsive Layout**: Designed to look great on desktops, tablets, and phones.
- **Tech Stack Logos**: Integration with Devicon to present a visual map of all technical skills.

## Getting Started

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed, and then execute the following commands in the terminal:

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Visit the `localhost` URL provided in the terminal (usually `http://localhost:5173`) to view the application in your browser.

## Customization

- **Profile Picture**: Replace `public/profile.jpg` with your own professional photo.
- **Content Updates**: Open `index.html` to modify text, project links, or add new experiences.
- **Style Overrides**: Tweak the colors inside the `:root` variables located at the top of `src/style.css` to instantly change the theme.

## Architecture

This project was bootstrapped using Vite (Vanilla template) to ensure ultra-fast hot module replacement during development and minimal configuration. Pure HTML, CSS, and vanilla JS (`main.js`) were used to keep the portfolio lightweight and performant.
